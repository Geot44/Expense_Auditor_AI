# Policy-First Expense Auditor 🛡️💰

An AI-driven compliance engine designed to automate corporate expense auditing. This project uses Large Language Models (LLMs) to cross-reference employee receipts against complex company policy documents.

## 🚀 Mid-Week Status (Phase 1 Complete)
- [x] Initial Project Architecture
- [x] Streamlit UI Prototype (Upload Portal)
- [x] Integration with Gemini 1.5 Flash for OCR/Extraction
- [ ] Policy Parsing Engine (In Progress)
- [ ] Compliance Scoring Logic (In Progress)

- **Intelligence:** Google Gemini API
- **Language:** Python 3.12
